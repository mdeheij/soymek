# SOYMEK
Artisanal soy milk calculator and process guide.

## Overview
A utility to calculate water/sugar ratios and batch sizes for home-brewed soy milk. Designed for small-batch production with specific safety limits for Sakura rice cookers.

## Features
- **Precision Ratios:** Dynamic calculation of water/sugar based on bean weight and concentration.
- **Batch Management:** Calculates batch counts and provides overflow warnings for rice cooker capacities.
- **Process Guide:** Integrated steps for the "raw filter" (pre-boil) methodology.

## Technical Details
- **Stack:** Vanilla JavaScript, CSS3.
- **Design:** Minimalist, high-contrast OLED-optimized interface.
- **Dependencies:** None.
